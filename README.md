# Genji Passkey

Next.js Web3 starter with passkey auth.

Live demo: **https://w3pk.w3hc.org**

- [Next.js](https://nextjs.org/) `v16.0.0`
- [w3pk](https://github.com/w3hc/w3pk) `v0.7.6`
- [Ethers](https://docs.ethers.org/) `v6.13.4`
- [Chakra UI](https://chakra-ui.com/) `v2.10.9`

## Install

```bash
pnpm i
```

## Run

Create a `.env` file:

```bash
cp .env.template .env
```

Then:

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
